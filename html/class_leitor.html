Descrição detalhada

Responsável por armazenar informações do leitor.

Os seguintes serviços estão disponíveis ao leitor: apresentar dados do usuário, editar dados do usuário, excluir conta, listar nomes dos vocabulários controlados, apresentar dados de vocabulário controlado, consultar termo, consultar definição de termo.
